계산기 제작 이론

 

자료 구조 스택(LIFO)을 이용한 계산기 제작 원리

연산자 우선 순위: ['-,+', '%,/,*', '^']: 뺄셈, 덧셈 < 나머지연산자, 나눗셈, 곱셈 < 거듭제곱

 

수식은 사람이 보통 사용하는 중위식 표현이 있고,

컴퓨터가 계산하는데 용이도록 연산자 우선 순위를 고려한 후위식이 있다.

 

계산기를 만들려면 입력받은 중위식을 후위식으로 변환해야 한다.

 

수식 예)

중위식: 중위식은 연산자가 피연산자들 중간에 위치한다.
2 * ( 3 + 5 )

후위식으로 변환 원리 3가지:
괄호는 출력하지 않는다.
먼저 기본적으로 기억할 것은 피연산자는 스택에 넣지 않고 출력하며
(가 들어오면 무조건 스택에 push,
연산자는 스택이 비어있을 경우 무조건 push,
연산자는 스택이 비어있지 않을 경우, 스택에 있는 연산자와 우선 순위 비교가 필요하다.
만약 현재 연산자 우선 순위가 크다면 pop 없이 push한다.
만약 스택에 들어있는 연산자의 우선 순위가 같거나 높다면 스택에 있는 연산자는 pop하여 출력하고
현재 연산자를 push해야 한다.
수식이 끝나면 스택이 빌 때까지 pop 하여 출력한다.

  1. 높은 연산자는 push, 낮은 연산자는 높은 연산자를 pop한 후 push, 피연산자는 출력.
  2. '(' 연산자가 들어오면 무조건 push
  3. ')' 연산자가 들어오면 맞는 쌍의 '('를 찾기 위해 '(' 나올 때까지 스택 데이터를 계속 pop...

연산자 *는 push
연산자 (는 push

후위식: 후위식은 연산자가 피연산자들 뒤에 위치한다.
2 3 5 + *

2

3

5

+

*

 

후위식을 계산하는 원리 2가지: 만약 연산자일 경우 피연산자 2개를 pop 해야 함.

  1. 피연산자가 들어오면 push
  2. 연산자가 들어오면 스택에 들어있는 피연산자 2개를 pop 수행하여 스택에 push.
    첫 번째 데이터가 두 번째 피연산자가 되고, 두 번째 데이터가 첫 번째 피연산자가 된다.
    연산 마친 뒤, 스택에 남아있는 하나의 피연산자가 연산결과 수행.

즉,
피연산자 2를 push
피연산자 3을 push
피연산자 5를 push
연산자 +를 만나면 스택에 들어있는 2개의 피연산자 5, 3을 pop
3+5를 계산: 8
연산자 *를 만나면 스택에 들어있는 2개의 피연산자 8, 2를 pop
2*8을 계산: 16

계산 결과값: 16
